Support

Please note, that the FIFE version of Unknown Horizons is no longer supported. You can ask for help about crashes, but it is unlikely that we can do anything about it.

We are working on a Godot version, but that one isn't playable yet.

Get in contact with the team using our Discord server. This is the main channel for contacting the team.

Get in contact with the team using our IRC channel on irc://irc.libera.chat/#unknown-horizons (Web IRC).

Leave us a message on the forum.

Leave a report at our bug tracker on GitHub for the new Godot version or the legacy FIFE version.

For any issues regarding the legacy version, please verify first if your problem is not already answered in our FAQ.
