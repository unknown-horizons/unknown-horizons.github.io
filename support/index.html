Support

Hello, if you have a problem with Unknown Horizons you are in the correct place to get help and solve your problem.

The FIFE version of Unknown Horizons is no longer supported. You can try to ask help about crashes, but it is unlikely that we can do anything about it.

We are working on a Godot version, but this is not playable yet.

Get in contact with the team directly using our Discord Server. This is the main method of contacting the team.

Get in contact with the team directly using our IRC Channel on irc://irc.libera.chat/#unknown-horizons

Get in contact with the team directly using our Web IRC

Leave us a message on the Forum

Leave a bugreport on our Bug Tracker at Github. Keep in mind that this version of Unknown Horizons is not supported anymore (we don' have developers for this) so we probably won't be able to fix it.
